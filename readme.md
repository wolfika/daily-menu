# daily-menu [![Build Status](https://travis-ci.org/wolfika/daily-menu.svg?branch=1.0.0-dev)](https://travis-ci.org/wolfika/daily-menu)

> Scraper utility tool to fetch daily menus.


## Prerequisites

* Node.js >=12
* npm


## Install

```
$ npm install -g daily-menu
```
